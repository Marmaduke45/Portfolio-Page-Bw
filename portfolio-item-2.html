Movie Watchlist

With the OMDB API

The movie watchlist allows a user to create a list of movies to watch so you can spend less time deciding the show or movie, and more time enjoying watching them.

This was a very fun project as I love movies, but who doesn't right? Along the way, I learned a lot about setting complex data in localStorage, and error handling as the OMDB database requests would often give errors if the user miss-spelt anything.

The hardest part was saving the movie data in localStorage so when you refresh the page, you can keep your watchlist.
