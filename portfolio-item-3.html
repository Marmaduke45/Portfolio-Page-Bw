Quizical

Trivia app

This is a quiz app built with the Open Trivia database and React. There's 3 difficulties to challenge anyone.

This project taught me a ton of stuff about single page applications with react including state management, dynamic rendering, REST api use, props, and context.

Though, there were some challenges I had to overcome such as saving the index of the correct answers despite them being shuffled, which meant I couldn't save the index of the correct answers but rather had to turn to saving the values of them. This made the app a bit more complicated to develop.
